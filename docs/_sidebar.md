## 技术栈

- **后端**: Spring Boot 4.0.1, Java 25
- **前端**: Vue 3 (3.5.22), Vuetify 3 (3.10.5)
- **数据库**: PostgreSQL, JPA/Hibernate
- **插件系统**: PF4J + Spring Boot Integration
- **证书管理**: ACME4J (3.5.0)
- **构建工具**: Vite 7 (7.1.5), Maven
- **依赖管理**: Hypersistence Utils 3.14.1
